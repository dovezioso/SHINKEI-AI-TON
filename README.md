# 神経 Shinkei AI Agent — TON Network

A professional Japanese-themed AI Agent web application built on The Open Network (TON) blockchain.

## Features

- **TON Wallet Integration** — Connect via Tonkeeper, TonHub, OpenMask, or MyTonWallet
- **AI Agent Chat** — Interactive demo agent with knowledge of TON ecosystem
- **Japanese Aesthetic** — Elegant design with Seigaiha patterns, kanji typography, and vermillion accents
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Animated UI** — Scroll reveals, particle effects, counter animations

## Quick Start

1. Clone this repo
2. Open `index.html` in your browser, or deploy to GitHub Pages
3. Update `tonconnect-manifest.json` with your actual URLs

## Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Push this code to the `main` branch
3. Go to Settings → Pages → Source: Deploy from branch → `main` / `root`
4. Update the manifest URL in `tonconnect-manifest.json`

## TON Connect Setup

For production wallet connections, update `tonconnect-manifest.json`:
- `url` — Your deployed site URL
- `iconUrl` — Your app icon (192x192 PNG)

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- TON Connect SDK
- Google Fonts (Noto Serif JP, Zen Kaku Gothic New)
